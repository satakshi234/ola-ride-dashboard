# ola-ride-dashboard
# 🚖 Ola Booking Dashboard Analysis

This project is based on analyzing ride booking data similar to Ola. The goal is to extract meaningful insights and visualize them using a dashboard.

# 📊 Dataset Features:

Booking ID, Date & Time

Booking Status (Completed, Cancelled, etc.)

Customer & Driver details

Pickup & Drop Locations

Ride Distance & Booking Value

Cancellation Reasons

Ratings (Customer & Driver)

# 📈 Key Insights:

Total bookings and revenue analysis

Cancellation trends (Customer vs Driver)

Ride distance distribution

Payment method analysis

Customer and driver rating analysis

# 🛠️ Tools Used:

Excel / Power BI / SQL (jo tumne use kiya ho)

Data Cleaning & Data Analysis
